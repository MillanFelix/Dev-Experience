## Sobre o Projeto

Este é o repositório do website desenvolvido para a empresa **TalentoFlow** no âmbito da unidade curricular de [nome da disciplina / curso].  
O objetivo é criar um site moderno, responsivo e profissional que represente a identidade da TalentoFlow, uma empresa especializada em *gestão de talento, recrutamento inteligente, desenvolvimento de carreiras e soluções de RH tech*.

### Funcionalidades Principais Implementadas / Planeadas

- Página Inicial (Hero section + Destaques + Call-to-Action)
- Sobre Nós / A Empresa
- Serviços (ex: Recrutamento, Assessment, Coaching de Carreira, Plataforma de Talentos)
- Página de Vagas / Oportunidades de Emprego
- Blog / Artigos (opcional – se houver tempo)
- Contactos + Formulário de contacto
- Área de Login / Área Privada (mockup ou simples, se aplicável)
- Design 100% responsivo (mobile-first)
- Animações suaves e interatividade (se possível com JS)

### Tecnologias Utilizadas

- **HTML5** + **CSS3** (Flexbox / Grid / Variáveis CSS)
- **JavaScript** (vanilla ou com pequenas bibliotecas)
- **Framework CSS** (escolham um):
  - Bootstrap 5
  - Tailwind CSS
  - Ou puro com custom styles
- **Deploy**:
  - GitHub Pages (recomendado para simplicidade)
  - Vercel / Netlify (muito recomendado – mais profissional e rápido)
- Ferramentas de desenvolvimento:
  - Git + GitHub
  - VS Code (ou editor à escolha)
  - Figma (para prototipagem – se tiverem)
